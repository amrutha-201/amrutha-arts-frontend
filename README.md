# 🎨 Amrutha Arts Frontend

## Overview

Amrutha Arts Frontend is a React-based web application designed to showcase handmade artworks through a modern and responsive user interface. The platform allows users to explore artwork collections, view artwork details, add available paintings to a cart, place painting orders, and submit custom artwork requests.

## Features

* Browse artwork gallery
* Search artworks by title or medium
* View detailed artwork information
* Add available artworks to cart
* Place painting orders
* Submit custom artwork requests
* Display artwork availability status
* Responsive design for desktop and mobile devices
* Admin dashboard integration

## Tech Stack

* React.js
* React Router
* Axios
* Bootstrap
* JavaScript
* HTML5
* CSS3

## Backend Integration

The frontend communicates with a Node.js, Express.js, and MongoDB backend using REST APIs to fetch artworks, manage orders, and update artwork availability.
